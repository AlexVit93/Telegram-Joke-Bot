# Telegram Joke Bot

Простой Telegram-бот, который отправляет случайные анекдоты, спарсенные с сайта.

## Описание

Бот парсит анекдоты с сайта **anekdotov.net** и выдаёт их пользователю по запросу.
После каждого запроса анекдот удаляется из списка, чтобы не повторяться.

---

## Функционал

- Парсинг анекдотов с сайта
- Перемешивание списка шуток
- Отправка анекдота по любому числовому сообщению (`1–9`)
- Простой текстовый интерфейс в Telegram
- Автоматическое обновление очереди шуток (без повторов)

## Как запустить

### 1. Клонировать проект

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 2. Установить зависимости

```bash
pip install pyTelegramBotAPI requests beautifulsoup4
```

---

### 3. Добавить токен бота

В файле `main.py` добавьте токен:

```python
TOKEN = 'YOUR_TELEGRAM_BOT_TOKEN'
```

---

### 4. Запустить бота

```bash
python main.py
```

---

## Как работает

1. При запуске бот загружает анекдоты с сайта:

   ```
   http://anekdotov.net/anekdot/
   ```

2. Все анекдоты сохраняются в список

3. Пользователь отправляет любое число (1–9)

4. Бот отправляет следующий анекдот из списка

---

## Используемые библиотеки

- `telebot` (pyTelegramBotAPI) — Telegram Bot API
- `requests` — HTTP-запросы
- `beautifulsoup4` — парсинг HTML
- `random` — перемешивание списка

---

## Важно

- Сайт-источник может менять HTML-структуру — парсер может сломаться
- При исчерпании списка бот перестанет выдавать анекдоты
